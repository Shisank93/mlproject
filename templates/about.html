About the Project

Understanding the Machine Learning architecture behind student mathematical forecasting.

Model Objective

The primary goal of this predictive engine is to analyze the correlation between student demographic background variables, socio-economic context, and standard preparatory scores to forecast mathematics outcomes.

By standardizing incoming numerical markers and dynamically encoding categorical labels, the model matches new profiles against historical assessment structures to predict performance accurately.

Features Scored

  • Demographics: Student gender and race/ethnicity grouping classifications.
  • Socio-Economic Factors: Parental level of education (e.g. associate, bachelor, master) and enrollment in standard or free/reduced lunch support plans.
  • Preparatory Work: Completion status of test preparation courses.
  • Adjacent Scores: Reading and writing test score parameters.
Tech Specifications
Core Algorithm Linear Regression Ensemble
Backend Platform Flask (Python 3.12)
Frontend Architecture Jinja2, Bootstrap 5, Vanilla CSS/JS
Data Processing StandardScaler & OneHotEncoder (Scikit-Learn)
